# Sistema Inteligente de Asistencia QR

## Descripción
Sistema inteligente de gestión y control de asistencia escolar mediante códigos QR, integración con Telegram y generación de reportes automatizados.

El sistema permite gestionar estudiantes, asistencia, horarios, notificaciones y reportes institucionales.

---

# Roles del sistema

## 1. Docente
Funciones:
- Inicio de sesión
- Registrar estudiantes
- Gestionar estudiantes
- Escanear códigos QR
- Registrar asistencia de entrada y salida
- Compartir códigos QR
- Compartir enlaces de vinculación con bot de Telegram
- Gestionar horarios
- Visualizar asistencias
- Generar reportes
- Exportar PDF y Excel

## 2. Administrador
Funciones:
- Gestión global del sistema
- Gestión de docentes
- Gestión de estudiantes
- Gestión de horarios
- Supervisión de asistencias
- Gestión de reportes
- Configuración institucional

---

# Integraciones

## Telegram Bot
El sistema integra un bot de Telegram para:
- Vincular padres de familia
- Enviar notificaciones automáticas
- Informar entrada y salida de estudiantes
- Alertas de asistencia

---

# Reportes Inteligentes

El sistema incluye:

- Generación de reportes automáticos
- Exportación PDF
